## What's This?

TinyToolbox Corporation is a **Paperclip-methodology** AI company with:
- **5 executives** (CEO + 4 VPs)
- **4 departments** (Product, Growth, Revenue, Content)
- **Daily heartbeat** at 7am-10:45am
- **Email reports** to the Board (skipartai@gmail.com)

## Quick Start

### View Current Status
```bash

# Check approval queue

# Check auto-action log
```

| Time | What Happens |

| File | Purpose |
|------|---------|
| `ORG-CHART.md` | Full org structure, KPI definitions, escalation rules |
| `KPI-DASHBOARD.md` | Current metrics vs targets |
| `ESCALATION-RULES.md` | What CEO can auto-do vs Board approval |
| `APPROVAL-QUEUE.md` | Items waiting for Board sign-off |
| `AUTO-ACTION-LOG.md` | Actions CEO took autonomously |

## CEO Powers (No Approval Needed)

- Fix typos and broken links
- Adjust agent schedules <2 hours
- Create regression tests
- Merge fixes that pass build
- Update dependencies

## Requires Board (Daniel) Approval

- New affiliate partnerships
- Site-wide design changes
- Add/remove agents
- Spend >$50/month
- Revenue-impacting changes

## How to Approve/Deny Items

Edit `APPROVAL-QUEUE.md`:
```markdown

- **Status:** PENDING
- **Board Decision:** вң… APPROVED  вҶҗ Add this
- **Decision Date:** 2026-03-29
```
